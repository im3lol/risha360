# Reesha AI Agents - IMPLEMENTATION.md

## 1. Project Overview

Reesha is a Saudi-first platform that connects influencers, artists, actors, creators, restaurants, advertising agencies, brands, and companies for marketing campaigns.

The current platform already exists.  
This implementation focuses only on building the AI Agent layer responsible for:

1. Discovering influencers and artists from Instagram, TikTok, and public web sources.
2. Collecting and enriching creator data.
3. Scoring and qualifying leads.
4. Storing leads in an external Supabase-powered dashboard.
5. Assigning leads to AI Sales Agents.
6. Preparing and managing outreach through Reesha official accounts.
7. Tracking conversations, registration status, and conversion rates.

## 3. Core Strategy

Reesha should not wait for creators to register manually.

Instead, Reesha should build a proactive acquisition engine:

```txt
Discover creators
↓
Analyze them
↓
Score them
↓
Store them
↓
Contact them
↓
Convince them to register
↓
Track conversion
```

The main asset is not only the platform.

The main asset is:

```txt
A clean, scored, continuously updated database of Saudi influencers and artists.
```

## 4.4 Scraping and Collection Stack

Available tools:

```txt
Apify
Crawlee
Firecrawl
Crawl4AI
Scrapling
Playwright
```

Recommended usage:

### Apify
Use for fast MVP collection.

Best for:

- Quick Instagram/TikTok actor testing
- Rapid proof of concept
- Early lead collection

Do not depend on Apify as the only long-term solution because it can become expensive at scale.

### Crawlee
Use as the main long-term crawler engine.

Best for:

- Queue management
- Retry logic
- Session handling
- Proxy handling
- Large crawling jobs
- Scalable scraping architecture

### Playwright
Use when pages require browser rendering.

Best for:

- JavaScript-heavy pages
- TikTok profile rendering
- Instagram public profile rendering
- Screenshots
- Human-like browser automation

### Scrapling
Use for fast extraction after HTML is loaded.

Best for:

- Parsing profile pages
- Extracting bio text
- Extracting links
- Extracting emails or visible contact data
- Extracting structured text blocks

### Firecrawl
Use for external link enrichment.

Best for:

- Linktree pages
- Creator websites
- Portfolio pages
- Agency pages
- Contact pages

### Crawl4AI
Use for AI-ready page conversion.

Best for:

- Turning pages into markdown
- Extracting structured information with LLM assistance
- Reading creator websites and media kits
- Summarizing long pages

## 6. Agent System

## 6.1 Discovery Director Agent

### Role
The main brain that directs all discovery and scraping.

It does not scrape directly.

It decides:

- What to search for
- Which platform to target
- Which category needs more leads
- Which city needs more coverage
- Which hashtags should be expanded
- When to stop a search
- When to create new crawler jobs

### Input
```json
{
  "market": "Saudi Arabia",
  "categories": ["food", "lifestyle", "fashion"],
  "min_followers": 20000,
  "target_count": 5000
}
```

### Output
```json
{
  "search_batches": [
    {
      "platform": "instagram",
      "category": "food",
      "city": "Riyadh",
      "queries": ["#مطاعم_الرياض", "#riyadhfood", "Riyadh food blogger"]
    }
  ]
}
```

## 6.12 AI Sales Outreach Agent

### Role
Writes personalized messages and suggests replies.

### Rules
- Never guarantee campaigns.
- Never guarantee income.
- Always mention free registration.
- Keep messages short.
- Personalize based on creator category.
- Use Arabic or English based on creator language.
- Escalate complex business questions to human.

### First Arabic Message
```txt
أهلًا {creator_name} 👋

فريق ريشة لاحظ محتواك على {platform}، وشايفين إنك مناسب تكون ضمن شبكة صناع المحتوى والفنانين على منصتنا في السعودية.

ريشة منصة بتساعد الشركات، المطاعم، الوكالات الإعلانية، والبراندات يوصلوا للمؤثرين والفنانين المناسبين لحملاتهم.

التسجيل مجاني، وبيخليك تظهر للجهات اللي تبحث عن صناع محتوى لحملات إعلانية.

هل تحب أرسل لك رابط التسجيل؟
```

### First English Message
```txt
Hi {creator_name} 👋

We are from Reesha, a Saudi platform connecting creators, influencers, and artists with brands, restaurants, agencies, and companies for marketing campaigns.

We came across your content on {platform} and believe your profile could be a strong fit for our creator network.

Registration is free and helps your profile become visible to businesses looking for creators for campaigns.

Would you like me to send you the signup link?
```

## 14. Compliance and Safety Rules

The system must:

- Avoid spam behavior.
- Avoid aggressive automation through social accounts.
- Use human approval in MVP.
- Store only relevant business acquisition data.
- Respect opt-out requests.
- Mark rejected leads properly.
- Avoid repeated messaging after rejection.
- Avoid guaranteeing income or campaigns.
- Avoid collecting non-public sensitive information.
- Use official Reesha accounts only for outreach.
- Keep outreach professional and transparent.
